Project 2: Number Guessing Game

As part of my Software Development Internship at SkillCraft Technology, I successfully completed Project 2 – Number Guessing Game using Java.

This project generates a random number and challenges the user to guess it. The program compares the user's guess with the generated number and provides feedback indicating whether the guess is too high, too low, or correct. The project helped me strengthen my understanding of Java fundamentals, user input handling, conditional statements, and random number generation.

Key Features:

Random number generation
User input handling using Scanner
Conditional logic for guess evaluation
Interactive command-line interface
Beginner-friendly Java implementation
